Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat struktur dasar dari halaman web. HTML menggunakan tag-tag seperti <h1>, <p>, dan <img> untuk menyusun elemen dalam sebuah halaman.

HTML bersifat statis, artinya tidak bisa merespons interaksi pengguna secara langsung. Tapi tanpa HTML, website tidak akan memiliki kerangka apapun.

Contoh Struktur HTML:

<html>
  <head><title>Judul Halaman</title></head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini paragraf pertama.</p>
  </body>
</html>

HTML juga bisa digunakan untuk membuat tabel, form input, daftar, dan banyak lagi.

Pengenalan CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan atau gaya dari elemen HTML. CSS memisahkan antara konten (HTML) dan desain (warna, layout, font, animasi).

Dengan CSS, halaman web bisa terlihat lebih menarik dan responsif di berbagai perangkat seperti laptop, tablet, dan smartphone.

Contoh Penggunaan CSS:

h1 {
  color: blue;
  font-size: 32px;
  text-align: center;
}

CSS bisa ditulis dalam file terpisah (.css) dan dihubungkan ke HTML menggunakan <link> di bagian <head>.

Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JavaScript, pengguna bisa berinteraksi dengan elemen di halaman seperti tombol, form, gambar, dll.

Contoh fitur yang dibuat dengan JavaScript: validasi form, animasi, kalkulator, popup, hingga aplikasi seperti game dan chat langsung.

Contoh JavaScript:

document.getElementById("demo").innerText = "Halo dari JavaScript!";

JavaScript biasanya diletakkan di file .js terpisah dan dihubungkan ke HTML melalui tag <script src="script.js"></script>.

Perbedaan HTML, CSS, dan JavaScript

  • HTML → Membuat struktur dasar halaman web (kerangka).
  • CSS → Mengatur tampilan visual dari elemen HTML (desain).
  • JavaScript → Menambahkan logika dan interaksi (dinamis).

Analogi: Jika website adalah tubuh manusia:

  • HTML = Tulang dan organ (struktur)
  • CSS = Kulit dan pakaian (tampilan luar)
  • JavaScript = Otak dan saraf (interaksi dan reaksi)
